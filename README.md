# 🚀 Agent Orbit

**A persistent memory AI coding agent with tools and voice-ready architecture.**

Built with Grok, LangGraph, and clean Python structure.

### Features
- Persistent memory (remembers across sessions)
- Tool use (file creation, git, shell, Trello)
- Clean credential management
- Ready for Gradio UI + voice

python -m venv venv
source venv/bin/activate    # On Windows: venv\Scripts\activate
