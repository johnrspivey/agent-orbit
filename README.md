# 🚀 Agent Orbit

**Persistent Memory + Tool-using AI Developer Agent**

Built with Grok, LangGraph, and clean architecture.

### Features
- Persistent memory across sessions
- Tool integration (Trello, Git, file editing, shell)
- Clean credential management
- Ready for voice interface

### Quick Start
1. Clone the repo
2. `pip install -r requirements.txt`
3. Add your keys to `.env` (see `.env.example` when we add it)
4. `python main.py`
